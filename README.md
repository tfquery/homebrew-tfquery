# Homebrew Tap for tfctl

# With pre-tapping.

```bash
brew tap tfctl/tfctl

# Initial install.
brew install tfctl

# Regular update.
brew update
brew upgrade tfctl
```

# Without pre-tapping.

```bash
# Initial install.
brew install tfctl/tfctl/tfctl

# Regular update.
brew update
brew upgrade tfctl/tfctl/tfctl
```
